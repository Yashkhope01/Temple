# Hero Background Image

Please add your temple background image here with the filename: `hero-background.jpg`

**Supported formats:**
- .jpg / .jpeg
- .png
- .webp

**Recommended specifications:**
- Size: 1920x1080 pixels or larger
- Format: JPG (for best compression)
- Quality: High quality (the component will optimize it)

**How to add:**
1. Place your image file in this `public/images/` folder
2. Name it exactly: `hero-background.jpg`
3. The image will automatically appear as the home page background

**Note:** If no image is found, the site will show an error. Make sure the file exists at `public/images/hero-background.jpg`
